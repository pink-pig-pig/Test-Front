=======
项目下载和运行
----

- 拉取项目代码
```bash
git clone https://github.com/pink-pig-pig/common-qa-front.git
cd kuaishou-ll-front
```

- 安装依赖
```
yarn install
```

- 开发模式运行
```
quasar dev
```

- 编译项目
```
quasar build
```

- Lints and fixes files
```
yarn run lint
```
